Since sparvk is a portable distribution of debian, python comes pre installed as a part of a package. This makes sparvk an ideal platform to learn python on. You can simply plug in a sparvk ready bootable drive and load it in a live mode to start your session. Open up a console do a bit of python programming and switch off when you're done.

Every new boot gives you a clean slate to work with. Great if you are trying out something new. With enough practice you'll be ready to write python programs of your own.

How to access python shell from sparvk?

  1. Open terminal
  2. Type python
  3. Hit enter

You'll be in a python shell now. Try a executing a simple command like help() first to get a hang of things before executing more complex programs.

If you're are new to python learn python in 15 minutes is an excellent resource to get comfortable with it. Python is a general purpose programming language that can be used for web development, scripting, machine learning and anything else you can think of. Python is an excellent introductory programming language. The concepts that you learn with python can be carried over to other programming languages. sparvk makes programming in python easy.
